# w2ui-dashboard

Template for a basic w2ui dashboard. It is bundled with a set of tools, libraries and features to  make it ready to use and industrialized.

## Feature set

* Support of Typescript
* Support of Webpack
  * asset copy
  * JS Bundling
* Support of web server through webpack
* W2UI Library
* ESLint
